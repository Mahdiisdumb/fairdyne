# Fairdyne (aka Unfairdyne)

A **simulation of Undyne** from Undertale — you know, the fishy boss who loves to chase you around.  

Despite the name “Fairdyne,” this is **totally not fair**. It’s a fun little simulator where you can watch, experiment with, or mess around with Undyne’s attack patterns and behaviors.

## What This Is

- A simulation of Undyne’s combat mechanics  
- Fully playable / interactive (depending on your build)  
- Named **Fairdyne** for fun, but don’t be fooled — she’s not fair  

It’s perfect for fans who want to see Undyne’s attacks in action or test dodging strategies.

## Features

- Undyne’s attack patterns modeled in code  
- Simple UI / visualizations  
- Easy to explore, tweak, or experiment with
